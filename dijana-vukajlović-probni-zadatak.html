PARAGRAF

HTML (HyperText Markup Language) i CSS (Cascading Style Sheets) su dva ključna elementa koja čine temelje moderne web stranice, a njihov nastanak usko je povezan s razvojem World Wide Weba. HTML je izumio Tim Berners-Lee 1989. godine dok je radio u CERN-u. Prvobitna ideja bila je stvoriti način za povezivanje dokumenata putem hiperteksta, što je omogućilo korisnicima da "putuju" između stranica klikom na linkove. HTML je omogućio strukturiranje sadržaja na webu, koristeći oznake (tagove) za označavanje elemenata kao što su naslovi, odlomci, slike i poveznice. CSS je, s druge strane, nastao kao odgovor na potrebu za odvojenim stiliziranjem web stranica, jer HTML nije bio dizajniran za upravljanje izgledom i dizajnom stranica. CSS je predstavljen 1996. godine od strane Håkon Wium Lie-a, dok je radio na projektu World Wide Web Consortium (W3C). CSS omogućava web dizajnerima da definiraju izgled stranica (boje, fontove, raspored, itd.) ne mijenjajući osnovnu strukturu HTML dokumenta. Ovaj odvojeni pristup ubrzo je postao standard, jer je omogućio fleksibilnost, lakšu promjenu dizajna i bolju optimizaciju za različite uređaje. Zajedno, HTML i CSS omogućuju izradu funkcionalnih i vizualno privlačnih web stranica, a njihov razvoj i usklađivanje s novim tehnologijama stalno traje, omogućujući još sofisticiranije i dinamičnije web aplikacije.
